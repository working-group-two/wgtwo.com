Software Engineer - Media Processing

Permanent employee, full-time - Stockholm
The team and your role We are looking for an experienced software engineer to join our Voice team. The Voice team is constantly improving our cloud native media infrastructure to offer a more resilient, scalable and best in class audio and video experience. We are also working on APIs to expose media streams crossing mobile core networks to enhance end user experience. If you have a demonstrated track record of developing cloud or media processing software, we would love to have you join us in this effort. 

Most of our components are in-house developed in Java/Kotlin, Go, or Erlang, and are deployed on our self-managed Kubernetes cluster running on AWS. We are however language agnostic and believe in using the right tool for the job. 

We are firmly rooted in having people own their own work. This means that you are allowed a lot of freedom when it comes to planning and which features to build. However, our engineers who develop the code are also responsible for operating the code once in production. We adhere to principles that include the use of continuous integration, continuous delivery, microservices and infrastructure as code. 

Qualities and experiences we believe you have:
  • 5+ years of C++. 
  • Proficiency in a programming language of choice.
  • Complex distributed systems or real time systems.
  • Scalable cloud services (AWS, or other cloud infrastructure providers). 
  • Caring about the code quality.
  • It is an added bonus if you have knowledge of Java, Rust, WebRTC, Gstreamer or Kubernetes, but it is not required.
