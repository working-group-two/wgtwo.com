# STP

A Signal Transfer Point (STP) is a node in an SS7 network that routes
signaling messages based on their destination point code in the SS7
network.

## Capabilities

- Protocol support (M2PA+MTP3, M3UA, SCCP, MAP, CAPv2)
- Congestion control
- Firewall
